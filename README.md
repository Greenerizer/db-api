## Getting Started

### Prerequisites

Run this on your terminal.
* npm
  ```sh
  npm init
  ```
  
* express
  ```sh
  npm install express
  ```
  
* nodemon
  ```sh
  npm install --save-dev nodemon
  ```
  
* cors
  ```sh
  npm install cors
  ```
  
* dotenv
  ```sh
  npm install dotenv
  ```
  
* body-parser
  ```sh
  npm install body-parser
  ```
  
* mysql
  ```sh
  npm install mysql
  ```
